## ゲーム概要

キラキラ女子高生のるーちゃんはSNS「ゆにってぃー」に複数の裏アカを持っています。どのアカウントも友達には見せられないポストばかり。アカウントを間違えて投稿してしまったら大変です！

## ゲームシステム

### メイン

- タイムラインに様々なポストが流れてくる
- プレイヤーは複数のアカウントを切り替え可能
- アカウントを変えてもタイムラインは変わらない。つまり、全てのアカウントに関わるポストが一つのタイムラインにまとめて流れてくる
- 投稿したいポストが、投稿したいものリストにどんどん溜まっていく。リストの各内容を適切なアカウントに切り替えて投稿すると加点
- 返信できるポストもある。返信は、投稿したいものリストとは別に複数の返信内容が表示され、正しい返信を選ぶと加点
- 正しいアカウントで正しいポストにいいねをすると加点
- リポストすると加点されるポストを正しいアカウントでリポストすると加点
- 間違ったアカウントで投稿やいいね、リポストをするとゲームオーバー
- タイムラインには広告ポストも流れてくる。これに触れるとゲームオーバー
- 制限時間内により大きなスコアを狙う

### 難易度

- 3段階から選べる
- 難易度が上がるごとに裏垢の数が増える
- 裏垢の数は最大3つ

### ランキング

- 最高難易度のスコアをunityroomのランキングで表示

## グラフィック

単一のSceneで全てのシーンを賄う

### ゲームシーン

- 中央にスマホに映るSNSを模した画面。タイムライン、投稿フィールド、アカウント切り替え、ハンバーガーメニュー(一時停止し設定画面を表示)などがある
- 右側に投稿したいものリストがある
- 左側にスコアと残り時間、動くキャラクターがいる
- 投稿したいものリストのコンテンツを投稿フィールドにドラッグアンドドロップすることで投稿
- ポストをクリックすると返信画面が表示される。右側に返信案が表示される
- ゲームオーバーまたはタイムアップで画面にスコア、ツイートボタン、リトライ、戻るボタンがある

### セレクトシーン

- 右側にスマホがあり、その画面に各難易度のボタンがある。ボタンには最高スコアが書いてある
- 左側にキャラクターがおり、各難易度の説明をする

### タイトルシーン

- SNSの新規登録画面みたいなものが画面いっぱいに映される
- スタートボタンでセレクトシーンに移行
- 初回はチュートリアルが流れる

## 必要素材

優先度が低い ≠ 必要ない

### 優先度高

- スマホ
    - 画面が透過されている(つまりフレームのみ)/されていないの2種
    - 画面部分の短い方が1080pxだと嬉しい
- ゲーム画面の背景
    - 部屋の中の絵とか風景画とかそういうものである必要は無く、それよりか[割とこの背景みたいなのを想像している(アニメーションさせられたらアツい)](https://youtu.be/dX7dhADuobI?si=9hpHplhiD_mzM8Rx)
    - が、他にもっと良いアイデアがあれば
- 画面左に置くキャラのイラスト
    - パーツを分けて出力
    - 表情差分, ポーズ差分あり

### 優先度中

- アイコン
    - 各アカウントのもの + モブ用いくつか
    - 500×500px程度の円形
- ツイートに添付される画像
    - どんなツイートを用意するかによる
- 広告ツイートの画像
    - 胡散臭いと面白そう
    - 脱毛, きぬた歯科インプラント, ...
    - 3枚ほどあったら嬉しい

### 優先度低

- タイトルロゴ
- ゲームアイコン
- ゆにってぃーのロゴ
- リザルト画面で出すキャラのイラスト
    - ミスらずタイムアップ(成功)とやらかしてゲームオーバー(失敗)の2通り
