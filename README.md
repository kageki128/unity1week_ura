## ゲームシステム

### メイン

- タイムラインに様々なポストが流れてくる
- プレイヤーは複数のアカウントを切り替え可能
- アカウントを変えてもタイムラインは変わらない。つまり、全てのアカウントに関わるポストが一つのタイムラインにまとめて流れてくる
- 投稿したいポストが、投稿したいものリストにどんどん溜まっていく。リストの各内容を適切なアカウントに切り替えて投稿すると加点
- 正しいアカウントで正しいポストにいいね、リポストをすると加点
- 間違ったアカウントで投稿やいいね、リポストをするとゲームオーバー
- タイムラインには広告ポストも流れてくる。いいねやリポストで減点
- 制限時間内により大きなスコアを狙う

### 難易度

- 3段階から選べる
- 難易度が上がるごとに裏垢の数や広告の数、流量が増える
- 裏垢の数は最大3つ

### ランキング

- 最高難易度のスコアと全難易度総合得点をunityroomのランキングで表示

## グラフィック

単一のSceneで全てのシーンを賄う

### ゲームシーン

- 中央にスマホに映るSNSを模した画面。タイムライン、投稿フィールド、アカウント切り替え、ハンバーガーメニュー(一時停止し設定画面を表示)などがある
- 右側に投稿したいものリストがある
- 左側にスコアと残り時間、動くキャラクターがいる
- 投稿したいものリストのコンテンツを投稿フィールドにドラッグアンドドロップすることで投稿
- ポストをクリックすると返信画面が表示される。右側に返信案が表示される
- ゲームオーバーまたはタイムアップで画面にスコア、ツイートボタン、リトライ、戻るボタンがある

### セレクトシーン

- 右側にスマホがあり、その画面に各難易度のボタンがある。ボタンには最高スコアが書いてある
- 左側にキャラクターがおり、各難易度の説明をする

### タイトルシーン

- SNSの新規登録画面みたいなものが画面いっぱいに映される
- スタートボタンでセレクトシーンに移行
- 初回はチュートリアルが流れる

## アカウントとポストの詳細

### ルール

- アカウント(以下、垢)は、自分が属するコミュニティを1つ持つ。
- 各垢がするポストは、自分の属するコミュニティだと明確に分かる内容でなくてはならない
- 各垢の名前からも、なんとなく所属するコミュニティが見分けられることが望ましい
- 各垢は、コミュニティ内でリプライなどで交流をしてもよい。逆に言うと、コミュニティを超えて交流してはならない。
- プレイヤー垢同士も、同一人物なのでもちろんやり取りをしてはならない。ただし、自分自身にリプを送るということはできる。
- 各垢は、後述する画像をポストに添付してもよい。ただし、この画像は汎用的なもので、誰でも何度でも使用してよい(専用画像は別)。
- 公序良俗に反する内容はNG
- 広告として、1つのポストのみ投稿する、胡散臭い広告垢が2つ存在する(ダイエットサプリ、セミナー)
- プレイヤーの投稿のいいね、リポスト数は0(ゲーム中で投稿するため、投稿直後だから)
- アカウントはAcounts.csv、ポストはPosts.csvに記入すること
- モブ垢のアイコン画像は、Mob0~6の中から選んで使うこと。
